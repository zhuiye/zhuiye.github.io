React Ref

Posted on 2019-12-14

基本使用

某些情况下,我们需要获取组件的实例,或者 react 元素的 Dom 节点,,这个时候就得用到 ref,关于 ref 的基本使用方法有两种:

  1. this.ref=createRef()
  2. <ReactDom ref={(ref)=>{ref}}/>

基本使用 : https://zh-hans.reactjs.org/docs/refs-and-the-dom.html
